# Changelog

All notable changes to the Eleutherios OpenClaw skill will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-30

### Added
- Initial release of Eleutherios skill for OpenClaw
- MCP integration via Streamable HTTP transport
- Tools: analyze_topic, get_perspectives, assess_source, get_claim_context, list_domains
- Documentation with usage examples
- Support for Linux, macOS, and Windows (WSL2)

### Prerequisites
- Eleutherios running locally via Docker
- MCP server on port 8100
